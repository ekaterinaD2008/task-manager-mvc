# Task Manager

**Автор:** Днепровская Екатерина Андреевна

Консольное приложение для управления задачами на Python.
Использует ООП, паттерн MVC, стек, очередь, JSON.

## Как запустить
1. Скачай ZIP-архив: зелёная кнопка Code → Download ZIP
2. Распакуй архив в любую папку
3. Открой командную строку в этой папке
4. Выполни команду: python main.py

## Возможности
- Добавление, редактирование, удаление задач
- Приоритеты: Low, Medium, High
- Статусы: To Do, In Progress, Done
- Очередь задач по приоритету
- Стек отмены последнего действия (Undo)
- Фильтрация по статусу и приоритету
- Сохранение и загрузка в JSON
- Проверка корректности ввода

## Структура проекта
- task.py — класс Task (модель одной задачи)
- model.py — класс TaskManager (управление задачами)
- view.py — класс View (интерфейс пользователя)
- controller.py — класс MenuController (обработка команд)
- main.py — точка входа в приложение
